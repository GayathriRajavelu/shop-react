
# Shop React

A simple e-commerce application built with React.

## Features

- Browse products
- Add items to cart
- View cart summary
- Responsive design

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

Runs the app in development mode. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Build

```bash
npm run build
```

Builds the app for production.

## Technologies

- React
- JavaScript
- TailwindCSS
